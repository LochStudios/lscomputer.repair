What we bring

Diagnostic kit, common spare parts, USB recovery tools, network test gear and a portable workbench mat. If we need a part we don't carry, we order it then and there.

What's covered

Diagnostic, configuration, software fixes, basic hardware swaps. Larger jobs (full builds, motherboard work) we'll usually take back to the workshop and return the machine when done.
